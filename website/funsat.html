Funsat

Funsat (mnemonic: "functional sat solver") is a Haskell SAT solver whose goal is to facilitate convenient embedding in other applications. Current features include two-watched literals, conflict-directed learning, non-chronological backtracking, a VSIDS-like dynamic variable ordering, restarts, and unsatisfiable core generation. It was developed using The Glorious Glasgow Haskell Compiler 6.8.2, but should work with 6.8.3. It is released under the LGPL.

Funsat solves SAT instances encoded in conjunctive normal form using the DIMACS CNF format (in my understanding, the most common/popular format). For example, SATLIB provides a bunch of SAT instances one could try. The code produces both a binary (for standalone solving) and a Haskell library (for integrating with other code). It is released under a BSD license (see file LICENSE in the repository for details).

Obtaining

Release 0.5 is available from Hackage: http://hackage.haskell.org/cgi-bin/hackage-scripts/package/funsat. If you have cabal-install installed, this means you can simply type:
$ cabal install funsat

Funsat is also available as a git repo:

$ git clone http://churn.ath.cx/funsat.git
The repository contains the funsat code as well as many SAT instances for benchmarking. It should be in a usable state. (I push changes to it when I intend others to be able to see them.)

Dependencies

Funsat depends on a few tools, all of which are available from hackage.
  • parse-dimacs: A parser for DIMACS CNF files
  • bitset: A pure bit set library

Installing

Funsat is cabalised, and by default produces a library and a binary. To buld, type:
$ ghc --make Setup.hs
$ ./Setup configure
$ ./Setup build
